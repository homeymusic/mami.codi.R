Fourier Transform
================

``` r
f = c(1, 2-1i, -1i, -1+2i)
fft_slow(f)
#> [1]  2+0i -2-2i  0-2i  4+4i
```

``` r
c_sound = 343
chord = tibble::tibble(
  f = 100*1:10,
  w = 2 * pi * f,
  l = c_sound / f,
  k = (2 * pi) / l
```
